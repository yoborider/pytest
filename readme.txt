Un cours intensif concis sur Python pour rafraîchir vos connaissances de base.
Vous avez étudié les structures de données Python les plus importantes et comment les utiliser dans des exemples de code.
Vous avez appris à contrôler le flux d'exécution du programme à l'aide des instructions if-elif-else, ainsi que des boucles while et for.
Vous avez revu les types de données de base en Python - booléens, entiers, flottants,
et string, et vous avez vu quelles opérations et fonctions intégrées sont couramment
utilisées. La plupart des extraits de code dans la pratique et les algorithmes non triviaux sont construits
autour de types de conteneurs plus puissants tels que les listes, les piles, les ensembles et les dictionnaires.
En étudiant les exemples donnés, vous avez appris à ajouter, supprimer, insérer et réorganiser des éléments.
Vous avez également découvert les opérateurs d'appartenance et de la compréhension des listes : une méthode intégrée
efficace et puissante pour créer des listes de manière programmatique en Python.
Enfin, vous avez appris à connaître les fonctions et comment les définir (y compris la fonction lambda anonyme).
